# SlackStream

**Awareness without interruption. Slack only when it matters.**

SlackStream is a passive Slack awareness tool designed for people who value deep focus. It lets you stay informed about the conversations that matter most, without keeping Slack open or constantly breaking flow.

> SlackStream does not replace Slack.
> It decides when Slack is worth opening.

---

## Why SlackStream Exists

Slack is powerful, but it creates a constant attention tax.

- Keeping Slack open disrupts flow
- Closing Slack risks missing something important
- Notifications create false urgency
- "Just checking Slack" often turns into context switching

SlackStream removes that tradeoff.

---

## Core Idea

SlackStream provides read‑only visibility into the Slack activity that matters most to you.

- You define VIP channels and priority conversations
- Slack stays closed while you work
- SlackStream passively surfaces high‑signal updates
- You decide if and when to log in to Slack

**No replies. No acknowledgements. No accidental engagement.**

If something needs your attention, you intentionally open Slack.

---

## Design Principles

SlackStream is opinionated by design.

| Principle | Description |
|---|---|
| **One‑way only** | SlackStream is inbound awareness, not interaction. |
| **Flow‑state safe** | No notifications designed to pull you in automatically. |
| **Intentional engagement** | Logging into Slack is a conscious decision, not a reflex. |
| **No noise** | Only channels and messages you explicitly care about. |
| **No false urgency** | Awareness first, action only when necessary. |

---

## How It Works

1. Configure VIP channels and priority conversations
2. Close Slack
3. Work heads‑down
4. SlackStream passively surfaces relevant updates
5. You choose:
   - Stay focused and ignore
   - Log in to Slack and engage

SlackStream is the gatekeeper. Slack is the destination.

---

## What SlackStream Is Not

- Not a replacement for Slack
- Not a chat client
- Not a notification spammer
- Not interactive by design

If you want to reply, react, or participate, Slack is still the right tool. SlackStream simply helps you decide when it is worth doing so.
